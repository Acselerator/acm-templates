# 后缀数组

- 后缀数组 $sa[i]$ ：将所有后缀排序后第 $i$
小的后缀的编号，也称编号数组

- 排名数组 $rk[i]$ :后缀 $i$ 的排名，是重要的辅助数组

有性质： $sa[rk[i]]=rk[sa[i]]=i$

![后缀数组示例](https://oi-wiki.org/string/images/sa1.png)

## 后缀数组 $O(nlog^2n)$ 求法

倍增：

用长度为 $w/2$ 的子串的排名，即 $rk_{w/2}[i]$ 和 $rk_{w/2}[i+w/2]$，作为排序的第一第二关键字，就可以对字符串 $s$ 的每个长度为 $w$ 的子串 $s[i\dots \min(i+w-1,\ n)]$ 进行排序，得到 $sa_w$ 和 $rk_w$。其中，类似字母序排序规则，当 $i+w>n$ 时，$rk_w[i+w]$ 视为无穷小

![倍增排序示意](https://oi-wiki.org/string/images/sa2.png)

[P3809 【模板】后缀排序](https://www.luogu.com.cn/problem/P3809)

```cpp
//洛谷P3809 完整代码
#include<iostream>
#include<cstring>
#include<algorithm>
using namespace std;
const int maxn=1e6+5;

char s[maxn];
int n,w,sa[maxn],rk[maxn<<1],oldrk[maxn<<1]; //后缀数组，排名数组

bool cmp(int x,int y){
    return rk[x]==rk[y]?rk[x+w]<rk[y+w]:rk[x]<rk[y]; //第一关键字相等时比较第二关键字rk[x+w]
}

int main(){
    int i,p;

    scanf("%s",s+1);
    n=strlen(s+1);
    for(int i=1;i<=n;i++){ //初始化
        sa[i]=i;
        rk[i]=s[i]; //直接令rk[i]为字符值，可使相同字符同值，便于比较
    }
    for(w=1;w<n;w<<=1){
        sort(sa+1,sa+n+1,cmp);
        memcpy(oldrk,rk,sizeof(rk));
        for(p=0,i=1;i<=n;i++){ //重新计算rk
            if(oldrk[sa[i]]==oldrk[sa[i-1]]&&
            oldrk[sa[i]+w]==oldrk[sa[i-1]+w]){ //实际上是用rk的比较代替了字符串字典序的比较
                rk[sa[i]]=p; 
            }else{
                rk[sa[i]]=++p;
            }
        }
    }
    for(int i=1;i<=n;i++)
        printf("%d ",sa[i]);
}
```
