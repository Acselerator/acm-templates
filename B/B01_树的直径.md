# 树的直径

[B4016 树的直径](https://www.luogu.com.cn/problem/B4016)

树上最远的两个节点之间的距离被称为树的直径（途经边的数量）

**方法一（两次DFS）：**

1. 任取一点作为起点x，找到距离该点最远的一个点y；

2. 再找到距离y最远的一点z，那么y、z之间的路径就是一条直径。

```cpp
//洛谷B4016完整代码
#include<iostream>
using namespace std;
const int maxn=1e5+5;

struct edge{
    int to,nxt;
}e[maxn<<1];
int head[maxn],tot;

int n;
int dep[maxn];
int fa[maxn];
int maxx,p=1;

void add(int x,int y){
    e[++tot].to=y;
    e[tot].nxt=head[x];
    head[x]=tot;
}

void dfs(int x){
    if(dep[x]>maxx){
        maxx=dep[x];
        p=x;
    }
    for(int i=head[x];i;i=e[i].nxt){
        int y=e[i].to;
        if(y==fa[x]) continue;
        fa[y]=x;
        dep[y]=dep[x]+1;
        dfs(y);
    }
}

int main(){
    scanf("%d",&n);
    for(int i=1;i<=n-1;i++){
        int x,y;
        scanf("%d %d",&x,&y);
        add(x,y);
        add(y,x);
    } 
    dfs(1);
    for(int i=1;i<=n;i++){
        dep[i]=0;
        fa[i]=0;
    }maxx=0;
    dfs(p);
    printf("%d\n",maxx);
}
```

**方法二（树形DP）：**

1. 任意选择一个点作为根节点，将整棵树“拎”起来，然后将路径分为若干类，每个点代表一类，每个点代表的集合是经过该点且该点是此路径最高点的所有路径的集合，属性是路径权值和最大值。

2. 在递归求解的过程中记录每个点所有子树的路径和最大值和次大值（可能相等）即可，最长路径就是两者之和。

```cpp
//洛谷B4016完整代码
#include<iostream>
using namespace std;
const int maxn=1e5+5;

struct edge{
    int to,nxt;
}e[maxn<<1];
int head[maxn],tot;

int n;
int fa[maxn];
int maxx,p=1;

void add(int x,int y){
    e[++tot].to=y;
    e[tot].nxt=head[x];
    head[x]=tot;
}

int dfs(int x){
    int d1=0,d2=0;
    for(int i=head[x];i;i=e[i].nxt){
        int y=e[i].to;
        if(y==fa[x]) continue;
        fa[y]=x;
        int d=dfs(y)+1;

        if(d>=d1) d2=d1,d1=d;
        else if(d>d2) d2=d;
    }
    maxx=max(maxx,d1+d2);
    return d1;
}

int main(){
    scanf("%d",&n);
    for(int i=1;i<=n-1;i++){
        int x,y;
        scanf("%d %d",&x,&y);
        add(x,y);
        add(y,x);
    } 
    dfs(1);
    printf("%d\n",maxx);
}
```
