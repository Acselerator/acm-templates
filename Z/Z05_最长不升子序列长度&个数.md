# 最长不升子序列长度&个数

[P1020 [NOIP 1999 提高组] 导弹拦截](https://www.luogu.com.cn/problem/P1020)

## 第一问

### dp做法

$dp_i$表示「对于前 i 个数，在选择第 i 个数的情况下，得到的单调不升子序列的长度最长是多少」

$dp_i = \max\left\{1, \max_{\substack{j < i \\ h(j) \geq h(i)}} (dp_j + 1)\right\}$

$O(n^2)$时间复杂度

```cpp
//洛谷P1020 第一问代码 dp做法
#include<iostream>
#include<sstream>
#include<cstring>
using namespace std;
const int maxn=1e5+5;

int dp[maxn];
int h[maxn],cnt;
int ans;
string s;

int main(){
    getline(cin,s);
    stringstream sin(s);
    int tmp;
    while(sin>>tmp){
        h[++cnt]=tmp;
    }
    for(int i=1;i<=cnt;i++){
        dp[i]=1;
        for(int j=1;j<i;j++){
            if(h[j]>=h[i])
                dp[i]=max(dp[i],dp[j]+1);
        }
        ans=max(ans,dp[i]);
    }
    printf("%d\n",ans);
}
```

### 二分做法

$f_i$表示「对于所有长度为 i 的单调不升子序列，它的最后一项的大小」的最大值。特别地，若不存在则 $f_i$=0。

$f_i$单调不增，我们要找到尽可能大的 x 满足 $h(i)≤f_x$
。考虑二分。

$O(nlogn)$时间复杂度

## 第二问

假设枚举到第 i 个导弹时，有 m 个系统。我们把这些系统的高度按照从小到大排列。容易发现我们就是要找到最小的 $g_x$
满足 $g_x \geq h_i$
​（与第一问相同，这是可以二分得到的），然后更新 $g_x$
的值。更新之后，$g$数组显然还是单调不增的，因此不用重新排序；如果找不到符合要求的导弹拦截系统，那就说明 $g_x \leq h_i$，直接在后头增加一个就行。

$O(nlogn)$时间复杂度

```cpp
//洛谷P1020 完整代码
#include<iostream>
#include<cstring>
using namespace std;
const int INF=2147483647;
const int maxn=1e5+5;

int n;
int H[maxn],F[maxn];

int main(){
	while(~scanf("%d",&H[++n]));
	n--;
	int t=0;
	memset(F,0,sizeof(F));
	F[0]=INF; //单调不增
	for(int i=1;i<=n;i++){
		int l=0,r=t+1;
		while(r-l>1){
			int mid=l+(r-l)/2;
			if(F[mid]>=H[i]) l=mid;
			else r=mid;
		}
		int x=l+1;
		if(x>t) t=x; //添
		F[x]=H[i];
	}
	printf("%d\n",t);
	t=0;
	memset(F,0,sizeof(F));
	F[0]=0; //单调不减
	for(int i=1;i<=n;i++){
		int l=0,r=t+1;
		while(r-l>1){
			int mid=l+(r-l)/2;
			if(F[mid]<H[i]) l=mid;
			else r=mid;
		}
		int x=l+1;
		if(x>t) t=x; //添
		F[x]=H[i];
	}
	printf("%d\n",t);
	return 0;
}
```

其实，第二问等价于计算**最长上升子序列长度**（严格上升，即后一项大于前一项）。这其实是 **Dilworth 定理**（将一个序列剖成若干个单调不升子序列的最小个数等于该序列最长上升子序列的个数）
