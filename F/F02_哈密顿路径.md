# 哈密顿路径

[P10447 最短 Hamilton 路径](https://www.luogu.com.cn/problem/P10447)

Hamilton路径：从图中的一个顶点出发，沿着边行走，经过图的每个顶点，且每个顶点仅访问一次

1. 哈密尔顿回路(Hamilton Loop)要求从起始点出发并能回到起始点，其路径是一个环。

2. 哈密尔顿路径(Hamilton Path)并不要求从起始点出发能够回到起始点，也就是说：起始顶点和终止顶点之间不要求有一条边。

对于哈密尔顿问题，目前并没有多项式级别的算法。我们只能在暴力破解的基础上，尽量去做到更多的优化，譬如回溯剪枝，记忆化搜索等，但是，还没有找到一种多项式级别的算法来解决哈密尔顿问题。

通常，这类问题也被称为 NP(Non-deterministic Polynomial)难问题。

```cpp
//洛谷P10447完整代码
//本题没有返回哈密顿路径的途中经过情况
#include<iostream>
#include<cstring>
using namespace std;
const int maxn=25;
const int maxm=1<<20;
const int inf=0x3f;

int n;
int a[maxn][maxn];
int f[maxm][maxn];
//f[i][j] 表示当前访问状态为 i 且最后一个访问的城市是 j 时的最短路径长度。
int main(){
    memset(f,inf,sizeof(f));
    f[1][0]=0;
    scanf("%d",&n);
    for(int i=0;i<n;i++){
        for(int j=0;j<n;j++){
            scanf("%d",&a[i][j]);
        }
    }
    for(int i=1;i<(1<<n);i++){ //状压dp，枚举状态
        for(int j=0;j<n;j++){ //枚举当前状态下的最后一个城市
            if(!((i>>j)&1)) continue; //如果j位为0，就跳过它
            for(int k=0;k<n;k++){
                if(((i^(1<<j))>>k)&1) //检查在移除 j 后的状态中， k 是否可能是 j 的前一个城市）
                    f[i][j]=min(f[i][j],f[i^(1<<j)][k]+a[k][j]);
            }
        }
    }
    printf("%d\n",f[(1<<n)-1][n-1]);
}
```
