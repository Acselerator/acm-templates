## 💡 写在前面

本仓库包含了我在此前并不算长的算法竞赛生涯中，汇集整理的一系列代码模板，由于我负责的方向的原因，所以没有涉及所有知识点。退役之后，我选择将它们上传到这里。

当初把这些文件集中存放在队长的服务器网盘上，本意是为了方便大家在训练和比赛时查阅使用，它们也确实给了我一些帮助。如今，并不奢求这套略带个人习惯的代码能帮到读者什么，权且当作一个记念吧。
